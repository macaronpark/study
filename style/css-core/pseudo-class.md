## 개념

요소가 특정 상태일 때 스타일을 부여함

- 예시
  - `:hover` 포인터를 올렸을 때
  - `:last-child` 형제 요소 중 제일 마지막 요소
  - `:last-of-type` 같은 태그명을 가진 형제 요소 중 마지막 요소

<br>

## `pseudo-class` vs `pseudo-element`

- 의사 클래스: 요소가 **특정 상태**일 때만 스타일 부여
- 의사 요소: 요소의 **특정 부분**에만 스타일 부여
