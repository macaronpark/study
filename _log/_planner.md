## 전체 계획 개요

### 1-2달차 (기초 탄탄히)

- **이론** (40%, 10시간): JS 핵심 개념 + 브라우저
- **코테** (40%, 10시간): 자료구조 기본 + 알고리즘 입문
- **사전과제** (20%, 5시간): 기본 CRUD 앱들

### 3-4달차 (실전 강화)

- **이론** (30%, 7.5시간): 면접 준비 + React 심화
- **코테** (30%, 7.5시간): Level 2 + 라이브코딩 연습
- **사전과제** (25%, 6시간): 복잡한 기능 구현
- **사이드프로젝트** (15%, 4시간): 포트폴리오 프로젝트
