## vscode

- 내부 터미널에서 oh-my-zsh 폰트가 깨질 때
  - settings(`Cmd/Ctrl + ,`) > `Terminal > Integrated: Font Family` > "MesloLGS NF"
- `code` 명령어 추가
  - `Cmd + Shift + P` > `Shell Command: Install 'code' command in PATH` > 터미널 재시작
- 커스텀 단축키 추가
  - `Cmd + Shift + P` > 원하는 명령어 입력 > 우측 톱니바퀴 클릭 후 키 설정
